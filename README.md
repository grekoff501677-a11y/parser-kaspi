# Kaspi Parser

Десктопный инструмент для парсинга товаров с [Kaspi.kz](https://kaspi.kz) с GUI на PyQt5.  
Разработан для мониторинга конкурентов и управления собственным ассортиментом на маркетплейсе.

---

## Возможности

| Вкладка | Что делает |
|---|---|
| **Парсер (Самовывоз)** | Обходит категорию (диски, колёса, автозапчасти), находит товары с опцией «Самовывоз», экспортирует в Excel |
| **Наш магазин** | Парсит товары нашего магазина (merchant id `30090572`), классифицирует по городу (Павлодар / Астана), экспортирует 2 файла |
| **Сравнение** | Сравнивает два Excel-отчёта за разные периоды — показывает исчезнувшие (🔴) и новые (🟢) позиции |
| **Дубли** | Находит и удаляет товары, которые одновременно есть в файлах Павлодара и Астаны |

### Ключевые детали
- **Batch-обработка**: товары открываются пакетами по 6 вкладок для скорости
- **Автофильтры**: автоматически применяет фильтры Kaspi (продавец, тип диска, комплектация) перед парсингом
- **Городская кука**: переключает город через cookie (`cityId`) — без смены настроек браузера вручную
- **Прогресс в реальном времени**: прогресс-бар + лог с таймштампами, обновляется по ходу обработки
- **Двойной клик по URL** в таблице → открывает страницу товара в браузере

## Установка

### Требования
- Python 3.10+
- Google Chrome (актуальная версия)

### Зависимости

```bash
pip install PyQt5 pandas openpyxl selenium webdriver-manager beautifulsoup4 lxml
```

### Запуск

```bash
python bot_opt.py
```

> При первом запуске `webdriver-manager` автоматически скачает ChromeDriver нужной версии.

## Использование

### Парсинг самовывоза

1. Вставьте URL категории Kaspi (или выберите из списка)
2. Выберите город (Астана / Павлодар)
3. Настройте автофильтры (продавец, тип, комплектация) при необходимости
4. Нажмите **«Фильтры»** — откроется браузер, примените дополнительные фильтры вручную
5. Нажмите **«Парсинг»** — парсер соберёт все ссылки, затем обойдёт каждый товар
6. После завершения — **«Экспорт в Excel»**

### Парсинг нашего магазина

1. Вкладка **«Наш магазин»**
2. Укажите URL категории, нажмите **«Открыть фильтры»**
3. В открывшемся браузере выберите нашего продавца и нужные фильтры
4. **«Парсинг»** → после завершения **«Экспорт (2 файла)»** — отдельно для Павлодара и Астаны

### Сравнение периодов

1. Вкладка **«Сравнение»**
2. Загрузите файл прошлого периода и файл текущего периода (`.xlsx`)
3. Нажмите **«Сравнить файлы»**
4. Красные строки — исчезнувшие товары, зелёные — новые
5. **«Экспорт результатов»** — сохраняет два листа: `Исчезнувшие` и `Новые`

> Формат файлов: обязательные столбцы — `Код товара`, `Наименование`, `URL`

### Удаление дублей

1. Загрузите файл Павлодара и файл Астаны
2. Нажмите **«Удалить дубли»** — из файла Астаны будут исключены товары, уже присутствующие в файле Павлодара
3. Сохраните очищенный файл (доп. листы: `Информация`, `Дубли`)

## Технический стек

- **GUI**: PyQt5 (QMainWindow, QTabWidget, QThread)
- **Браузерная автоматизация**: Selenium + webdriver-manager
- **Парсинг HTML**: BeautifulSoup4 + lxml
- **Работа с данными**: pandas, openpyxl
- **Многопоточность**: QThread с сигналами (`pyqtSignal`) для обновления UI без заморозки

---

## Известные ограничения

- Kaspi периодически меняет CSS-классы — XPath/CSS-селекторы могут потребовать обновления
- Скорость парсинга зависит от интернет-соединения и загруженности Kaspi (~6 вкладок / батч)
- Браузер должен быть открыт и авторизован в Kaspi Merchant Cabinet перед запуском парсинга нашего магазина
- Антибот-защита Kaspi: при слишком агрессивном парсинге возможна временная блокировка IP
